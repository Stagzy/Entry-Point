## Additional Assets Needed

### App Icon
- 1024x1024px icon for App Store
- Various sizes for device display

### App Store Metadata
- App name: "Entry Point - Fair Giveaways"
- Subtitle: "Verified Creators, Real Prizes"
- Keywords: giveaway, contest, win, prizes, verified, fair, creators
- Category: Entertainment
- Content rating: 17+ (due to gambling-like mechanics)

### Privacy Labels
- Email addresses (linked to user, app functionality)
- Name (linked to user, app functionality)  
- Financial info (linked to user, app functionality)
- Precise location (not linked, app functionality)
- Usage data (not linked, analytics)

### App Store Description
See `app-store-description.md` for full marketing copy.

## Screenshot Guidelines

1. **Use clean, representative data** - No placeholder text
2. **Show verification badges** - Highlight trust and safety
3. **Include real prize examples** - iPhone, gaming gear, cash
4. **Demonstrate mobile-first design** - Clean, modern UI
5. **Show social proof** - Recent winners, active community
6. **Highlight key differentiators** - AMOE compliance, trust tiers, verification

## Screenshot Sizes Required

### iPhone 6.7" (iPhone 14 Pro Max, 15 Pro Max)
- 1290 x 2796 pixels
- Required for App Store

### iPhone 6.5" (iPhone 11 Pro Max, XS Max)  
- 1242 x 2688 pixels
- Optional but recommended

### iPad Pro 12.9" (5th generation)
- 2048 x 2732 pixels  
- Required if supporting iPad

## Tools for Screenshots

1. **iOS Simulator** - Built into Xcode, pixel-perfect
2. **Device Screenshots** - Using real devices via Xcode
3. **Screenshot Framer** - Add device frames and backgrounds
4. **App Store Screenshot Generator** - Automate sizing
